WordPress Remote Manager Enhanced Users Edition with Email Support - Like ManageWP functionality.

== Description ==

WP Remote Manager Enhanced Users v3.2.0 Final provides comprehensive WordPress site management capabilities with enhanced user metadata and email support, similar to ManageWP functionality.

= Key Features =

* Enhanced User Metadata with Email Support
* Advanced Plugin Management with detailed metadata
* Theme Management with complete information
* Secure API Endpoints with authentication
* WordPress Core Update Detection
* ManageWP-like functionality
* Enhanced Security Features
* Real-time Status Monitoring
* SSL Detection and Reporting
* Memory and Disk Usage Monitoring
* Comprehensive Site Health Reporting

= API Endpoints =

The plugin provides both secure and legacy API endpoints:

* `/wp-json/wrm-secure/v1/status` - Site status and health information
* `/wp-json/wrm-secure/v1/updates` - Available updates for plugins, themes, and core
* `/wp-json/wrm-secure/v1/plugins` - Complete plugin list with metadata
* `/wp-json/wrm-secure/v1/themes` - Complete theme list with metadata
* `/wp-json/wrm-secure/v1/users` - Enhanced user data with email support

= Authentication =

All API endpoints require authentication using the API key generated by the plugin. The API key can be passed via:
* X-API-Key header
* api_key parameter

= Enhanced User Features =

* User email addresses included in API responses
* User roles and capabilities
* User metadata (first name, last name, description)
* User registration dates
* Complete user permissions mapping

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-remote-manager-enhanced-users` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to Settings -> Remote Manager Enhanced to configure the plugin
4. Copy the generated API key to your WordPress Maintenance Dashboard

== Frequently Asked Questions ==

= How do I get the API key? =

After activating the plugin, go to Settings -> Remote Manager Enhanced in your WordPress admin. The API key will be displayed there. You can also regenerate it if needed.

= Is this compatible with ManageWP? =

This plugin provides ManageWP-like functionality with enhanced features, but it's designed to work with WordPress Maintenance Dashboard applications.
